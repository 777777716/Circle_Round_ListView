# Circle_Round_ListView
循环复用的圆弧形ListView

# 当前Unity版本
5.3.5f1

# Demo展示
直接运行/_Scene/Demo.unity场景

# 文件说明

### RoundScroll.cs 
实现了圆弧形的ListView操作，其中主要实现了滑动操作、操作惯性、定点吸附

### RoundCircleList.cs 
实现了ListView中Item的复用逻辑

### RoundCircleItem.cs 
实现了ListView中Item的数据初始化、显示、隐藏、角度修正等逻辑

### RoundCircleItemData.cs 
定义了ListView中Item所需要显示的数据

### Test.cs
对ListView的数据进行了初始化
